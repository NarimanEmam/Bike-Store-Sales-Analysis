# 🚴 Bike Store Sales & Product Performance Analysis

## 📌 Project Overview
This project analyzes the Bike Store dataset to evaluate sales performance, profitability, and product demand using a complete ETL workflow in Excel.  
The goal of the analysis is to identify top-performing products, revenue trends, and profit margins to better understand business performance and product demand.

## 🔧 Tools & Process
- Power Query (ETL & Data Cleaning)
- Data Modeling
- KPI Development
- Pivot Tables
- Interactive Excel Dashboard

## 📊 Key Focus Areas
- Sales Revenue Trend
- Profit & Margin Analysis
- Product Category Performance
- Country-Level Sales Analysis
- Most Ordered Products & Sub-Categories
- Top Products by Revenue

## 📈 Business Questions Answered
- Which product categories generate the most revenue?
- Which products are ordered the most?
- Which countries generate the highest sales and profit?
- How does revenue change over time?
- Which products deliver the highest margin?

## 📂 Files Included
- Excel Dashboard File (BikeStore_Dashboard.xlsx)
- Raw Data Files (BikeStore_Data)
- Dashboard Screenshots (Screenshots/)

## 🎯 Outcome
This project strengthened my ability to transform raw sales data into structured insights and interactive dashboards, helping highlight key business patterns in product performance and revenue growth.
